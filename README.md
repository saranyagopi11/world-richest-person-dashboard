# 🌍 World Richest Persons Dashboard | Power BI

## 📌 Project Overview
This project presents an interactive Power BI dashboard that analyzes the world's richest persons using key business intelligence and data visualization techniques.

The dashboard provides insights into:
- Total Net Worth
- Country-wise billionaire distribution
- Industry analysis
- Source of wealth comparison
- Ranking analysis
- Global wealth distribution

This project demonstrates skills in:
- Data Cleaning
- Data Modeling
- Data Visualization
- Dashboard Design
- KPI Creation
- Business Intelligence
